# 引言

今天刷 b 站的时候发现了一个很有意思的项目，就是今天的主角 ----ZeroBot
它原版功能较少，我选了一个带插件的扩展版本
Github:https://github.com/FloatTech/ZeroBot-Plugin
配合 cqhttp 和云服务器可以实现 24 小时全天在线的 qq 机器人，
cqhttp 的 Github 仓库:https://github.com/Mrs4s/go-cqhttp
机器人 qq 号：3552676907

# qq 群内的使用方法

# 1. 邀请 QQ 机器人入群，先 @机器人 输入 "/ 响应", 然后等机器人回复之后就可以正常使用了，完整:

# @只因哥 / 响应.(如果没有反应可以多输入几次)

# 2. 机器人会不定时的出来交流，这个看情况，触发了事件就会有反应.

# 3. 一些目前可使用的命令 (都要 @一次)

例:@只因哥 群温度
以下省略@只因哥,"[]"内为自填内容,"#"后面是我写的注释,不要打上去.
早安
晚安
抽象翻译[]
翻转[] #[]内填英文,意思是把这段英文翻转过来,例如:翻转i love you ! = noʎ ǝʌol ᴉ
组cp[@xxx][@xxx] #发送一段关于他俩的暧昧文案
磕cp #后面跟两个人名与上面的类似
今日早报 #发送一段早报
小作文 #发送一段嘉然逆天小作文
发大病 #发一段与嘉然有关的病
运势
抽签#上面的运势和这个一样,都是发送今天的运势
一言#十个好句
随机图片#随机的一张二次元图片
随机galCG #随即一个galgame的CG

# 目前实现的特殊功能只有这些，你也可以说一些在此范围之外的，它有可能会正常回复你，也有可能会胡言乱语

# 这个项目我放在雨云的服务器上跑，兜里就 15 块也租不到什么很好的服务器，所以我选了一个美国的低配服务器，有些时候可能反应会比较慢，大家耐心等等

# 这个问题等我什么时候富裕了在解决 (应该是解决不了了)

# 先试行一个月，没出什么事再说.

# 实现过程

# 將这两个项目下载到本地之后，打开 finalshell 连接自己的服务器，这个服务器没必要用很好的，差不多的就差不多了，一定要是 linux 系统！！！

# 之后把这两个项目直接复制粘贴到服务器的 root 目录下，先运行一下 cqhttp, 选项选 2, 生成 config 文件后更改里面的内容，主要是 qq 号和密码，密码一定要写.

# 后把签名服务器的地址粘贴到 "sign-sever:" 就好了，这个签名服务器 cqhttp 的 latest release 下就有，完成之后保存重新运行一遍 cqhttp, 选 2, 进入网址打开网络检测

# 记录 ticket 复制粘贴回服务器终端，接着接收手机验证码，最后同时运行 cqhttp 和 zerobot 就行了.
